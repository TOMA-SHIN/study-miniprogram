# 学习打卡小程序

微信小程序云开发学习打卡应用。

## 功能

- 学习计划管理
- 每日打卡记录
- 学习统计与日历
- 云数据库与云函数
- 用户身份识别
- 图片上传与云存储
- 深色模式
- 分享好友

## 目录结构

```
cloudfunctions/    # 云函数
miniprogram/       # 小程序代码
db-init/           # 数据库初始化数据
.trae/specs/       # 项目方案文档
```
